┌─ FLUXO PRINCIPAL ─────────────────────────────────────────────────────────┐
│                                                                            │
│   1. Usuário faz LOGIN                                                    │
│      └─→ Backend verifica credenciais                                     │
│      └─→ Gera JWT token                                                   │
│      └─→ Frontend armazena token em localStorage                          │
│                                                                            │
│   2. Usuário faz UPLOAD DE VÍDEO                                          │
│      └─→ Upload arquivo + metadados                                       │
│      └─→ Backend valida                                                   │
│      └─→ FFmpeg processa (marca d'água)                                   │
│      └─→ Salva em /storage e registra em BD                               │
│                                                                            │
│   3. Usuário BUSCA & FILTRA                                               │
│      └─→ GET /videos?busca=termo&categoria=1                              │
│      └─→ Backend busca em BD (título, descrição, autor)                   │
│      └─→ Retorna lista paginada                                           │
│                                                                            │
│   4. Admin IMPORTA VIDEO DO YOUTUBE                                       │
│      └─→ POST /videos/:id/importar                                        │
│      └─→ Backend baixa com ytdl-core                                      │
│      └─→ Aplica marca d'água com FFmpeg                                   │
│      └─→ Salva no servidor (tipo muda para INTERNO)                       │
│      └─→ Vídeo disponível para streaming                                  │
